# 패스트 캠퍼스 10개 프로젝트로 완성하는 백엔드 웹개발

게시판 서비스는 강의와 공부 목적으로 만들어진 프로젝트입니다. 따라서 강의의 어떤 시점으로든 코드를 찾아가서 쉽게 관찰할 수 있도록 되어 있습니다. 특정 강의 시점의 소스코드를 좀 더 편리하게 확인하고 싶다면 릴리즈 탭을 확인해 보세요.


## 개발 환경
- Intellij IDEA Ultimate
- Java 17
- Gradle
- Spring Boot 2.7.15
